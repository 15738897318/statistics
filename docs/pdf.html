Function Reference: pdf

statistics: retval = pdf (name, X, …)

Return probability density function of name function for value x.

This is a wrapper around various namepdf and name_pdf functions. See the individual functions help to learn the signification of the arguments after x. Supported functions and corresponding number of additional arguments are:
