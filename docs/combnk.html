Function Reference: combnk

statistics: c = combnk (data, k)

Return all combinations of k elements in data.

Source Code: combnk

Example: 1

  

 c = combnk (1:5, 2);
 disp ("All pairs of integers between 1 and 5:");
 disp (c);
